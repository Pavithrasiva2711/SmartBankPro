 🏦 SmartBankPro – AI Powered Banking Management System

SmartBankPro is a secure and intelligent Banking Management System developed using **Python**, **Django**, **Django REST Framework**, and **PostgreSQL**. It provides banking operations through both a web interface and REST APIs, along with AI-powered assistance using the Gemini API.

🌐 Live Demo

**Live Website:** https://smartbankpro.onrender.com

 🚀 Features

* User Registration & Login
* Secure JWT Authentication
* Deposit Money
* Withdraw Money
* Money Transfer
* Transaction History
* Bank Statement Download (PDF)
* Bank Statement Download (Excel)
* User Profile Management
* Password Change
* AI Banking Chatbot (Gemini AI)
* REST APIs
* Swagger API Documentation
* PostgreSQL Database
* Responsive User Interface

 ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Pavithrasiva2711/SmartBankPro.git
```

Move into the project

```bash
cd SmartBankPro
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Apply migrations

```bash
python manage.py migrate
```

Run the project

```bash
python manage.py runserver
```
